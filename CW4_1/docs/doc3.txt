Запуск Java-приложения начинается с метода main, который выполняется в главном потоке.
Для создания своего потока унаследуйтесь от Thread или реализуйте Runnable.
Важно правильно обрабатывать исключения в потоках, например InterruptedException.
Потоки могут обмениваться данными через общие ресурсы, но нужна синхронизация.
Java 8 представила Stream API для параллельной обработки коллекций.
